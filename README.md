# portafolio2
Este trabajo es mi presentacion proyecto del segundo periodo en la clase de laboratorio 2 usando Ubuntu y los comandos git, que consite en tener 3 ramas, la master, portafolio y footer. Se trabaja en las distintas ramas hasta terminar la pagina web, luego se une con un comando llamado "git merge" con el podemos unir la rama footer y portafolio con la master para que el trabajo completo se quede en una rama.
